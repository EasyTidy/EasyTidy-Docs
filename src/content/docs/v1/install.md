## Build Manually

### Required Tools

* [Git](https://git-scm.com/)
* [.NET 8](https://dotnet.microsoft.com/zh-cn/download/dotnet)
* [Visual Studio 2022 Build Tools](https://aka.ms/vs/17/release/vs_BuildTools.exe)

### Build Steps

1. Clone the repository.
2. Install the Desktop development workload in the Visual Studio 2022 Build Tools.
3. Start the build.
