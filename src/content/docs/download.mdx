## 🔒 File Verification

To verify that your downloaded files are complete and have not been tampered with, download the checksum file and compute the SHA-256 hash:

- [checksums.txt](https://github.com/EasyTidy/EasyTidy-Releases/releases/latest/download/checksums.txt)

Then verify with:

```powershell
Get-FileHash EasyTidyPro_WithRuntime_Setup.exe -Algorithm SHA256
```
