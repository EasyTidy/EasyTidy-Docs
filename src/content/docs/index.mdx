## Features

<CardGrid stagger>
  <Card title="Auto File Classification" icon="list-format">
    Classify by name, directory, size, date, attributes, content, or regular expressions.
  </Card>
  <Card title="Real-Time Folder Monitoring" icon="magnifier">
    Monitor folder changes in real time and trigger automatic classification.
  </Card>
  <Card title="Scheduled Task Execution" icon="clock">
    Automatically run tasks on a set schedule based on matching conditions.
  </Card>
  <Card title="Flexible Rule System" icon="code-branch">
    Combine CRON expressions with multi-condition rules for complex classification needs.
  </Card>
  <Card title="Automated Workflows" icon="star">
    Chain multiple file operations together into automated workflows.
  </Card>
  <Card title="Local & Cloud Storage" icon="server">
    Run locally or extend via WebDAV and other cloud storage backends.
  </Card>
</CardGrid>
