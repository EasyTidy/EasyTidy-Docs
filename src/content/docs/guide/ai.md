Configure AI models to enable intelligent file classification, summarization, and other AI-powered features.

## Service Application

To use AI features, you need to obtain an API key from a supported AI provider. EasyTidy supports various AI services compatible with OpenAI API format.

## Adding an AI Service

1. Go to the AI model settings page.
2. Click "Add" to configure a new AI model.
3. Enter the model name, API endpoint, and API key.
4. Save the configuration.

## Setting a Default AI Model

After adding AI models, you can set one as the default. The default model will be used for:

- **AI Summary**: Summarize file content and output as PDF.
- **AI Classification**: Use natural language to classify and organize files.

:::tip
Different models can be selected for different tasks. The default model is used when no specific model is selected.
:::
