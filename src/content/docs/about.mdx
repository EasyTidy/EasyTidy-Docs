**EasyTidy Pro** is a modern, automated file organization and offline productivity tool built for Windows users.

## 🎯 Our Mission

In daily work and development, the desktop and downloads folder quickly fill up with all kinds of temporary files.

Manual organization is time-consuming and easy to forget, while long-term neglect impacts both productivity and mental clarity.

EasyTidy's goal is simple:

> **Make file organization as hands-off as possible.**

We want it to be:
- Lightweight enough
- Reliable enough
- And unobtrusive — never getting in your way

---

## 💎 Free Edition vs. Pro Edition

EasyTidy originated as an open-source project, so core foundational features remain free forever.

The free edition covers everyday file organization and basic automation needs.

The Pro edition is designed for higher-frequency, more complex use cases:

- Longer history retention and recovery capabilities
- More advanced automation orchestration
- Deeper cross-device and multi-source data management

### ⭐ Pro Edition Core Advantages

Pro is not about "removing limits" — it's about unlocking a complete file management toolkit:

- Cross-device tag and data sync
- Real-time file change awareness and auto-processing
- More complex automation workflow support
- Longer operation history traceability
- Stability designed for long-running environments

## 🙏 Acknowledgements & Open Source

EasyTidy Pro is built on the modern .NET ecosystem and relies on many outstanding open-source projects.

These projects have greatly improved development efficiency and product stability — our sincere thanks to them all.

### Core Dependencies

- .NET Runtime
- CommunityToolkit.MVVM (MVVM architecture)
- Serilog (structured logging)
- Quartz.NET (task scheduling)
- SharpCompress (compression support)
- WebDAVClient (remote storage access)
