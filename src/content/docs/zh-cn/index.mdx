## 特色功能

<CardGrid stagger>
  <Card title="自动文件分类" icon="list-format">
    按名称、目录、大小、日期、属性、内容或正则表达式分类
  </Card>
  <Card title="实时文件监控" icon="magnifier">
    实时对文件夹变动情况进行监控，自动分类操作
  </Card>
  <Card title="定时任务执行" icon="clock">

  </Card>
  <Card title="灵活规则系统" icon="code-branch">

  </Card>
  <Card title="自动化工作流" icon="star">
    将多个文件操作组合为自动流程
  </Card>
  <Card title="本地与云扩展" icon="server">
    支持本地执行与 WebDAV 等扩展存储
  </Card>
