**EasyTidy Pro** 是一款专为 Windows 用户打造的现代化、自动化文件整理与离线效率工具。

## 🎯 我们的初心

在日常工作与开发中，桌面与下载目录往往会迅速被各类临时文件占据。

手动整理既耗时又容易遗忘，而长期不整理又会影响效率与心智负担。

EasyTidy 的目标很简单：

> **让文件整理这件事，尽可能不再需要“手动处理”。**

我们希望它成为一个：
- 足够轻量
- 足够可靠
- 并且不会打扰用户的工具

---

## 💎 免费版与 Pro 版

EasyTidy 起源于开源项目，因此我们始终保留核心基础能力的免费使用。

免费版足以满足日常文件整理与基础自动化需求。

而 Pro 专业版，则面向更高频、更复杂的使用场景：

- 更长周期的历史记录与恢复能力
- 更复杂的自动化编排能力
- 更深度的跨设备与多源数据管理能力

### ⭐ Pro 专业版核心优势

Pro 并不是“去掉限制”，而是获得更完整的文件管理能力：

- 跨设备的标签与数据同步
- 实时文件变化感知与自动处理
- 更复杂的自动化工作流支持
- 更长周期的操作历史追溯能力
- 面向长期运行的稳定性设计

## 🙏 致谢与开源依赖

EasyTidy Pro 基于现代 .NET 生态构建，并依赖多个优秀的开源项目。

这些项目极大提升了开发效率与产品稳定性，在此表示感谢。

### 核心依赖生态

- .NET Runtime
- CommunityToolkit.MVVM（MVVM 架构支持）
- Serilog（结构化日志）
- Quartz.NET（任务调度）
- SharpCompress（压缩支持）
- WebDAVClient（远程存储访问）
