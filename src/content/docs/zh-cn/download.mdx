## 🔒 文件安全验证

为确认下载的文件完整且未被篡改，可下载校验文件并计算 SHA-256 哈希值：

- [checksums.txt](https://github.com/EasyTidy/EasyTidy-Releases/releases/latest/download/checksums.txt)

使用以下命令验证：

```powershell
Get-FileHash EasyTidyPro_WithRuntime_Setup.exe -Algorithm SHA256
```

:::tip[关于安全软件误报]
作为独立开发的效率工具，部分版本在下载或首次运行时可能会触发 Windows Defender 的「未知发布者」警告。请放心，EasyTidy Pro **纯净无毒，绝不上传任何用户隐私**。点击「仍要运行」即可安全使用。
如果您希望在运行前进行安全验证，我们建议并欢迎您使用第三方独立分析站点进行查杀。例如，您可以将文件上传至 [VirusTotal](https://www.virustotal.com/gui/home/upload) 或 [Hybrid Analysis](https://hybrid-analysis.com/) 进行分析。您也可以使用您个人信任的任何安全检测平台。在确认主流一线杀毒引擎均未报毒后，点击「仍要运行」即可安全使用。
:::
