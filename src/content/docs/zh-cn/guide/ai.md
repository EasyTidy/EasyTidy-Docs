配置 AI 模型以启用智能文件分类、总结和其他 AI 驱动的功能。

## 服务申请

要使用 AI 功能，需要从支持的 AI 服务商获取 API 密钥。EasyTidy 支持兼容 OpenAI API 格式的各种 AI 服务。

## 增加 AI 服务

1. 进入 AI 模型设置页面。
2. 点击"添加"配置新的 AI 模型。
3. 输入模型名称、API 端点地址和 API 密钥。
4. 保存配置。

## 设置默认 AI

添加 AI 模型后，可以设置其中一个为默认模型。默认模型将用于：

- **AI 总结**：总结文件内容并输出为 PDF 文件。
- **AI 分类**：使用自然语言进行分类并对文件进行整理。

:::tip
不同模型可用于不同任务。在未指定特定模型时，将使用默认模型。
:::
